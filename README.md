SnakeGame
====================
The snake game over led array.

It's required the library `LedControl` and `LiquidCrystal_I2C`

The code is prepared to work with:

* A set of 8 LED matrices (8x8) + MAX7219
* LCD Screen 1602 + I2C LCMI 602
* Arduino nano
* Joystick
